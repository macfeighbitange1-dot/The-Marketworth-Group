The Internet Connection

[ COURSE: AGENTIC SWARMS // MODULE 03 // TOOL INTEGRATION ]

A sovereign agent must be able to verify facts in real-time. Today, we grant our swarm the ability to Google Search and Scrape Websites. This turns a simple script into a competitive intelligence engine.

Step 1: Install the "Senses"

We use DuckDuckGo Search (because it’s free and requires no API keys) and CrewAI Tools for web scraping.

pip install duckduckgo-search crewai[tools]

Step 2: Import the Tools

We need to tell Python which "tools" are available for the agents to pick up and use.

from crewai_tools import WebsiteSearchTool
from langchain_community.tools import DuckDuckGoSearchRun

# Initialize the search tool
search_tool = DuckDuckGoSearchRun()
# Initialize a specific tool to read a website
web_tool = WebsiteSearchTool()

Step 3: Assigning Tools to Agents

Crucially, we don't give every tool to every agent. We give the search_tool to the Researcher, and the web_tool to the Analyst.

researcher = Agent(
  role='Market Scout',
  goal='Find the current price of Bitcoin in KES',
  backstory='Real-time financial data expert.',
  tools=[search_tool],
  llm=local_llama
)

Step-by-Step Execution Example

1. The Trigger: You ask for the weather in Nairobi.
2. The Thought: Agent realizes "I don't know the weather."
3. The Action: Agent calls DuckDuckGoSearchRun.
4. The Observation: Tool returns "24°C, Sunny."
5. The Final Answer: Agent tells you the current weather.
